About Our Project

Sign Language Recognition Technology

Project Overview

Project in B.E. at Chitkara University, we developed a Hand Gesture Recognition system that uses computer vision and machine learning to detect and classify hand gestures in real-time. This project allowed us to apply our knowledge of AI, computer vision, and software development to create a practical application.

Real-time Processing

We optimized our algorithms to process webcam video at 20+ frames per second on standard laptop hardware.

Machine Learning Model

We trained a convolutional neural network on a dataset of over 5,000 hand gesture images we collected ourselves.

Application Demo

We created a simple game controlled entirely by hand gestures to demonstrate the practical applications.

Why Use Hand Gesture Recognition

Hand gesture recognition technology offers numerous benefits across various domains. Our project explores several key applications that make this technology particularly valuable:

Accessibility

Enables users with mobility impairments or those who cannot use traditional input devices to interact with computers and smart devices more easily.

Interactive Gaming

Creates more immersive gaming experiences without requiring specialized hardware, using just a standard webcam for motion control.

Touchless Interfaces

Particularly valuable in public spaces, healthcare settings, or industrial environments where touching surfaces may be impractical or unhygienic.

Sign Language Translation

Has potential to bridge communication gaps between deaf and hearing communities by automatically translating sign language gestures.

Virtual Reality

Enhances VR experiences by allowing for natural hand interactions within virtual environments without requiring controllers.

Education

Creates interactive learning experiences that can help students engage with content through physical movement and interaction.

Technical Approach

Our project combines several technologies and approaches to create an effective hand gesture recognition system:

  • Image Processing Pipeline

    Using OpenCV for real-time video capture, hand region detection, and preprocessing

  • Neural Network Architecture

    A modified ResNet architecture implemented in TensorFlow with custom layers for gesture classification

  • Web Interface

    Frontend built with Bootstrap and JavaScript, backend using Flask to handle the ML model inference

  • Gesture Set

    Currently recognizes 10 different hand gestures including numbers 0-5, thumbs up/down, and victory sign
